# skripsi
skripsi menentukan rute terdekat dengan algoritma dijkstra dan a* dengan heuristic manhattan dan euclidean (studi kasus wilayah sekitar condong catur)
bahasa pemrograman
node js dengan framework express js
untuk template engine menggunakan hbs

cara install
import dulu file .sql dengan nama database sig
lalu jalankan npm index.js
buka localhost:8000/perbandingan
